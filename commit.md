## git commit

Команда **git commit *[опции]*** - создает новую фиксацию файлов добавленных в индекс.

```bash=
git commit -m "add index.html"
```
### Опции

`-m`

`--message`

* Содержит информацию о текщуих изменениях

[Подробная информация](https://git-scm.com/docs/git-commit)
